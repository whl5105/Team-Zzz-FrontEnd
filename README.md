## Development Setup
```bash

# Move to your workspace
cd your-workspace

# Clone this repository:
$ git clone https://github.com/whl5105/Team-Zzz-FrontEnd.git

# Enter in directory:
$ cd Team-Zzz-FrontEnd

# Install dependencies:
$ yarn install

# Run the development server:
$ yarn start

# Open the project in your browser
$ http://localhost:3000
```
